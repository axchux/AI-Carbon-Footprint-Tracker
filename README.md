# AI-Carbon-Footprint-Tracker

A simple web application to calculate your carbon footprint based on transportation and electricity usage.

## Features

- Calculate carbon emissions from kilometers traveled and electricity consumption
- Simple web interface
- Built with Flask

## Installation

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser to `http://localhost:5000`

## Usage

Enter the kilometers you travel and your electricity usage in kWh, then click Calculate to see your carbon footprint.
